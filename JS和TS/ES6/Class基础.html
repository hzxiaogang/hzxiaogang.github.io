1.简介 ​

1.1 类的由来 ​

JavaScript 语言中，生成实例对象的的传统方法是通过构造构造函数，ES6 提供了类似面向对象语言的写法。ES6 引入class(类)这个概念，作为对象的模板，通过class关键字定义类。

js
class Point {
  // 通过构造函数 定义实例属性 x，y
  // this 代表实例对象
  constructor(x, y) {
    this.x = x;
    this.y = y;
  }
  toString() {
    return `x:${x}，y:${y}`;
  }
}

构造函数的prototype属性，在 ES6 的类上继续存在。事实上，类的所有方法都定义在类的prototype属性上面。因此在类的实例上调用方法，其实就是在调用原型上的方法。

js
class Point {
  constructor(){//...}
  toString(){//...}
  toValue(){//...}
}
// 等同于
Point.prototype = {
  constructor(){//...},
  toString(){//...},
  toValue(){//...}
}

1.2 constructor 方法 ​

constructor()方法是类的默认方法，通过new命令生成对象实例时，自动调用该方法。一个类必须有constructor()方法，如果没有显示定义，ES6 会默认添加一个constructor()方法。constructor()方法默认返回实例对象（既this），我们也可以指定返回另外一个对象。

js
class Point {
  // 没有定义构造函数，ES6会默认添加一个
}
// 等同于
class Point {
  constructor() {}
}

1.3 getter 和 setter ​

在类的内部可以使用get和set关键字，对某个属性设置存值函数和取值函数，拦截该属性的存取行为。存值函数和取值函数是设置在属性的 Descriptor 对象上的。

js
class CustomHTMLElement {
  constructor(element) {
    this.element = element;
  }

  get html() {
    return this.element.innerHTML;
  }

  set html(value) {
    this.element.innerHTML = value;
  }
}

var descriptor = Object.getOwnPropertyDescriptor(
  CustomHTMLElement.prototype,
  "html"
);

"get" in descriptor; // true
"set" in descriptor; // true

2.静态方法 ​

所有在类中定义的方法，都会被实例继承。如果在一个方法前加上static关键字，这个方法就是静态方法。静态方法只能通过类来调用。如果静态方法包含this，这个this指的是类，而不是实例。

js
class Foo {
  static bar() {
    // 这里的 this 指的是 类 Foo
    this.baz();
  }
  static baz() {
    console.log("hello");
  }
  baz() {
    console.log("world");
  }
}
Foo.bar(); // hello
new Foo().baz(); // world

父类的静态方法可以被子类继承。只能通过子类名调用，不能通过子类实例调用。

js
class Father {
  static sayHello() {
    return "hello";
  }
}
class Child extends Father {
  static sayWorld() {
    // 静态方法也是可以从super对象上调用的
    return super.sayHello() + "，world";
  }
}

Child.sayHello(); // 'hello'
Child.sayWorld(); // 'hello，world'

4.静态属性 ​

静态属性指的是Class本身的属性，即Class.propName，而不是定义在实例对象(this)上的属性。下面的写法为Foo类定义了一个静态属性prop

js
// 老的写法

// 新的写法
class Foo2 {
  static prop = 2;
}
Foo2.prop; // 2

5. 静态块 ​

ES2022 引入了静态块，允许在类的内部设置一个代码块，在类生成时允许一次，主要作用是对静态属性进行初始化。静态块内部可以使用类名或this，指代当前类。每个类只能有一个静态块，在静态属性声明后运行。静态块的内部不能有return语句。

js
class Foo {
    static x = ...;
    static y;
    static z;
    // 静态块，初始化静态属性
    static {
        try{
            // 静态块的 this 指代当前类
            const obj = doSomethingWith (this.x)
            this.y = obj.y;
            this.z = obj.z;
        }
        catch{
            this.y = ...;
            this.z = ...;
        }
    }
}

6. new.target 属性 ​

new是从构造函数生成实例对象的命令。ES6 为new命令引入了new.target属性。该属性一般用于构造函数中，返回new命令作用于的那个构造函数。如果构造函数不是通过new命令或Reflect.constructor调用的，new.target会返回undefined。因此这个属性可以用来判断构造函数是怎么调用的。new.target只能用于函数内部或者类内部，否则会报错。

6.1 构造函数内部使用 ​

new.target在构造函数内部使用会返回构造函数

js
function Person(name) {
  if (new.target === Person) {
    this.name = name;
  } else {
    throw new Error("必须使用 new 命令生成实例");
  }
}
var person = new Person("张山"); // 正确
var notAPerson = Person.call(person, "张山"); // 报错

6.2 Class 内部使用 ​

new.target在 Class 内部使用，返回当前Class。但是当子类继承父类时，new.target会返回子类。

js
class Rectangle {
  constructor(length, width) {
    console.log(new.target === Rectangle);
    this.length = length;
    this.width = width;
  }
}

class Square extends Rectangle {
  constructor(length, width) {
    super(length, width);
  }
}

var obj1 = new Rectangle(3, 4); // 输出 true
var obj = new Square(3, 4); // 输出 false

  constructor(length, width) {
    super();
    // ...
  }
}

var x = new Shape(); // 报错
var y = new Rectangle(3, 4); // 正确
