# 数组Array的属性和方法总结

javaScript的 Array 对象是用于构造数组的全局对象,数组是类似以列表的高阶对象

# Array.length

Length 是 Array 的实例属性。返回或设置一个数组中的元素个数。

let color = ['red','orange','yellow','blue','pink'];  
color.length = 3; // ['red','orange','yellow']
color.length = 5; // ['red','orange','yellow',undefined,undefined]

# Array.prototype.concat()

concat() 方法用于合并两个或者多个数组。此方法不会更改现有数组，而是返回一个新数组。

var arr1 = [1,2,3]
var arr2 = ['a','b','c']
var arr3 = [4,5,6]
var list = arr1.concat(arr2,arr3)
console.log(list)
// [1,2,3,'a','b','c',4,5,6]

# Array.prototype.copyWithin()

copyWith() 方法浅复制数组的一部分到同一数组的另一个位置，并返回它，不会改变原数组的长度。

  1. 语法
    • arr.copyWithin(target[, start[, end]])
  2. 参数
    • target：0为基底的索引，复制序列到该位置。如果是负数，target 将从末尾开始计算
    • start ：0为基底的索引，开始复制元素的起始位置，如果是负数，start将从末尾计算
    • end ： 0为基低的索引,copyWithin将会拷贝到该位置，但不包括end这个位置的元素。如果是负数，end将从末尾开始计算。如果end被忽略，将会一直复制到数组结尾
    const array1 = ['a','b','c','d','e']
array1.copyWithin(0,3,4) // ['d','b','c','d','e']
array1.copyWithin(1,3) // ['d','d','e','d','e']

# Array.prototype.fill()

fill 方法用一个固定的值，填充一个数组从起始索引到终止索引内的全部元素。不包括终止索引。

  1. 语法
    • arr.fill(value[, start[, end]])
  2. 参数
    • value : 用来填充数组的值
    • start : 起始索引，可选，默认值为 0
    • end : 终止索引，默认值为 this.length
    const list = [1,2,3,4];
list.fill(0,2,4); // [1,2,0,0]
list.fill(6) // [6,6,6,6]

# Array.prototype.find()

find 方法返回数组中满足提供测试函数的第一个元素的值，否则返回 undefined

  1. 语法
    • arr.find(callback[, thisArg])
  2. 参数
    • callback 在数组每一项上执行的函数，接收三个参数
      1. element 当前遍历到的元素 可选
      2. index 当前遍历到的索引 可选
      3. array 数组本身 可选
    • thisArg 执行回调时 用作 this 的对象 可选
const array = [5,12,8,130,44]
const found = array.find(element=>element >10)
console.log(found) // 12  值

# Array.prototype.indexOf()

indexof 方法返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1

  1. 语法 arr.indexOf(searchElement[, fromIndex])
  2. 参数
  • searchElement 要查找的元素
  • fromIndex 开始查找的位置。 如果该索引值大于或等于数组长度，意味着不会在数组里查找，返回 -1。 如果索引值是一个负值，则将其作为数组末尾的一个抵消，既-1 表示从最后一个元素开始查找，-2 表示从倒数第二个元素开始查找，以此类推。索引为负值并不会改变其查找顺序，查找顺序任然是从前往后。如果抵消后的索引值仍小于0，则整个数组都会被查询。 默认值为0

# Array.prototype.lastIndexOf()

lastIndexOf 方法返回指定元素在数组中的最后一个索引，如果不存在则返回-1 。

  1. 语法
  • arr.lastIndexOf(searchElement[, fromIndex])
const animals = ['Dodo,'Tiger','Penguin','Dodo']
animals.lastIndexOf('Dodo')  // 3
animals.lastIndexOf('Tiger') // 1

# Array.prototype.includes()

includes 方法用来判断一个数组是否包含一个指定的值，包含则返回 true，不包含则返回 false

  1. 语法
    arr.includes(valueToFind[, fromIndex])
  2. 参数
  • valueToFind 需要查找的元素值
  • fromIndex 从fromIndex 索引处开始查找。 如果为负值，则按升序 array.lengt + fromIndex 的索引开始搜索 (既从末尾开始往前跳fromIndex的绝对值个索引，然后往后搜索)。默认值为0。 如果fromIndex 小于0 且 array.lengt + fromIndex 的值小于0 , 则从数组的其实位置开始搜索。
let list = [1,2,3,4]
list.includes(1)  // true
list.inclides(5) // false

# Array.prototype.push()

push 方法将一个或多个元素添加到数组的末尾，并返回该数组的新长度

  1. 语法 arr.push(element1, ..., elementN) 2.参数
  • elementN 被添加到数组末尾的元素
var sports = ['socer','baseball']
var total = sports.push('football','swimming')
console.log(total) // 4
console.log(sports) // ['socer','baseball','football','swimming']

像数组一样使用对象
push 方法具有通用性，该方法和 ***call()或apply()**一起使用时，可应用在类似数组的对象上。push方法根据 length 属性决定从哪里开始插入给定的值。如果length不能被转成一个数值，则插入的元素索引为0，包括length 不存在时。 当length 不存在时，将会创建它。

var obj = {
   length : 0,
   addElem : function addElem (elem){
       // obj.length is automatically increment
       // every time an element is added
       [].push.call(this,elem)
   }
};

obj.addElem({});
obj.addElem({});
console.log(obj.length) // 2
console.log(obj) // {0:{},1:{},length:2,addElem:f}

# Array.prototype.pop()

pop 方法从数组中删除最后一个元素，并返回该元素的值。此方法会更改数组的长度

  1. 语法 arr.pop()
const plants = ['broccoli', 'cauliflower', 'cabbage', 'kale', 'tomato']
console.log(plants.pop()) // 'tomato'
console.log(plants)  // ['broccoli', 'cauliflower', 'cabbage', 'kale',]

2
3

pop 方法有意具有通用性。该方法和 call() 或 apply() 一起使用时，可应用在类似数组的对象上。pop方法根据 length属性来确定最后一个元素的位置。如果不包含length属性或length属性不能被转成一个数值，会将length置为0，并返回undefined。
如果你在一个空数组上调用 pop()，它返回 undefined。

const firstElement = array.shift();

# Array.prototype.join()

join 方法将一个数组或（类数组对象）的所有元素连接成一个字符串并返回这个字符串。如果数组只有一个项目，那么将返回该项目而不使用分隔符。（先将元素转为字符串，然后用分隔符连接）

  1. 语法 arr.join([separator])
  2. 参数
  • separator 分隔符，可选。如果缺省，默认使用,分割。如果是空字符串""，则所有元素之间都没有任何字符。
var a = ['Wind', 'Rain', 'Fire'];
var myVar1 = a.join();      // myVar1的值变为"Wind,Rain,Fire"
var myVar2 = a.join(', ');  // myVar2的值变为"Wind, Rain, Fire"
var myVar3 = a.join(' + '); // myVar3的值变为"Wind + Rain + Fire"
var myVar4 = a.join('');    // myVar4的值变为"WindRainFire"

# Array.prototype.slice()

slice 方法用于截取数组。不会改变原数组

  1. 语法 arr.slice([begin[, end]])
  2. 参数
  • begin 截取数组的起始位置，可选。如果为负数，这表示从数组的倒数第几个元素开始截取，如slice(-2)表示提取原数组中的倒数第二个元素到最后一个元素(包含最后一个元素)。如果省略begin，则从0开始截取。如果begin超出原数组的索引范围，则返回空数组。
  • end 截取数组的结束位置（不包行end位置的元素），可选。如果为负数，则表示在倒数第几个元素结束截取。如果end省略，则一直截取到数组的结尾。如果end大于数组的长度，也会截取到数组的结尾。
var fruits = ['Banana', 'Orange', 'Lemon', 'Apple', 'Mango'];
var citrus = fruits.slice(1, 3);

// fruits contains ['Banana', 'Orange', 'Lemon', 'Apple', 'Mango']
// citrus contains ['Orange','Lemon']

# Array.prototype.splice()

splice 方法可以删除数组元素，替换数组元素，添加新的数组元素。 此方法会改变原数组。

  1. 语法 array.splice(start[, deleteCount[, item1[, itemN[, ...]]]])
  2. 参数
  • start 指定修改的开始位置(0记数)
    • 如果超出数组长度，则从数组末尾开始添加内容
    • 如果是负数，这表示从数组末位开始的第几位（从-1计数，这意味着-n是倒数第n个元素并且等价于array.length-n）；如果负数的绝对值大于数组的长度，则表示开始位置为第0位。
  • deleteCount 整数，表示要移除的数组元素的个数
    • 如果 deleteCount 被省略了，或者它的值大于等于array.length - start(也就是说，如果它大于或者等于start之后的所有元素的数量)，那么start之后数组的所有元素都会被删除。
    • 如果 deleteCount 是 0 或者负数，则不移除元素。这种情况下，至少应添加一个新元素。
  • itemN 要添加进数组的元素,从start 位置开始。如果不指定，则 splice() 将只删除数组元素。可选
var myFish = ['angel', 'clown', 'mandarin', 'sturgeon'];
var removed = myFish.splice(2, 0, 'drum', 'guitar');
// 运算后的 myFish: ["angel", "clown", "drum", "guitar", "mandarin", "sturgeon"]
// 被删除的元素: [], 没有元素被删除

# Array.isArray()

Array.isArray() 用于确定传递的值是否是一个数组，如果值是数组，则为true; 否则为false

  1. 语法
    Array.isArray(obj)
  2. 参数
    • obj 需要检测的值
// 下面的函数调用都返回 true
Array.isArray([]);
Array.isArray([1]);
Array.isArray(new Array());
Array.isArray(new Array('a', 'b', 'c', 'd'))
// 鲜为人知的事实：其实 Array.prototype 也是一个数组。
Array.isArray(Array.prototype);

// 下面的函数调用都返回 false
Array.isArray();
Array.isArray({});
Array.isArray(null);
Array.isArray(undefined);
Array.isArray(17);
Array.isArray('Array');
Array.isArray(true);
Array.isArray(new Uint8Array(32))
Array.isArray({ __proto__: Array.prototype });
